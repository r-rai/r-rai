I am an Integration Specialist with over 10 years of experience in enterprise application integration, specializing in on-premise and hybrid cloud integration solutions using the IBM webMethods product suite. I have a strong background in cloud integration, enabling businesses to leverage the benefits of cloud-based platforms while ensuring seamless connectivity and data flow across systems.

I have a proven ability for strategic planning, meticulous organization, and efficient prioritization. I can swiftly comprehend and assimilate new technologies to develop innovative business solutions.

### Recent Projects

**Digital Driven Experience (Stock Market)**
Led the integration efforts in revamping the digital infrastructure for a financial client, ensuring seamless connectivity between BPM, CRM, Equator, Order Streaming systems, and the front end through APIs and WebSockets. Designed and optimized a robust middleware platform, enabling efficient execution of critical financial services.

**SAG iPaaS Migration (Likewize)**
Led the SAG iPaaS Migration project, overseeing the seamless transition of existing application code/logic from On-Prem wM 9.12 to webMethods.io and MSR 10.7 using a hybrid approach. Focused on developing high-quality, scalable code tailored to project requirements while leveraging the full capabilities of each platform.

**Setting up high availability environment (Singapore based Retailer)**
This project included analysis of existing components, script generation, environment installation, code and configuration migration, testing support, and documentation. Also, did certificate installation to support secured (HTTPS) inbound and outbound communications from the application.

**Fourkites and Isotrak Integration (Major food retailer)**
This project was done to integrate TLM app with the third-party apps using REST API for a major food retailer. As a consultant, worked with Customer to get the requirements, provided estimates, and prepared mapping sheets. On the technical side, I did set up of new environment on the cloud and developed and unit-tested interfaces required for application Integration.

**Migration of SOAP to REST based APIs (Insurance Company)**
The purpose of the project was to migrate the existing SOAP based web services to REST based web services on the Enterprise Service Bus (ESB). This includes the development of new services, unit testing, defect fixing, and documentation.
